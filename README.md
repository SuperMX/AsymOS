# ⚡️ AsymOS: The Shadow Layer for Desktop Automation

> "The future isn't software; it's Agents."

AsymOS is an open-source framework that allows AI to "inhabit" your computer. Unlike traditional automation, AsymOS uses **Vision-based Reasoning** to navigate any UI, just like a human.

## 🚀 The Vision
In 2026, privacy and local control are everything. AsymOS lives on your machine, handles your "boring" tasks (like Realtor SEO audits or Student note organization), and stays out of the cloud.

## 🛠 Features (v0.1 - Early Alpha)
- **Computer Vision:** Takes real-time screenshots to "see" your desktop.
- **Agentic Logic:** Uses LLMs to decide where to click and what to type.
- **Asymmetrical Output:** Designed to do 10 hours of work in 10 minutes.

## 🏗 Built With
- **Python** (The core engine)
- **PyAutoGUI** (The "Hands")
- **OpenAI/Anthropic Vision APIs** (The "Brain")

## 📦 Dependencies

Install the required Python packages:

```pip install pyautogui pillow requests ollama```

Download the LLaMA 3.2 Vision model (requires ~7.8GB):

```ollama pull llama3.2-vision```

⚠️ Note: Ensure you have at least 8GB of free storage for the model.

## 🚀 Running the Project

Start the agent with:

```python asymos.py```
